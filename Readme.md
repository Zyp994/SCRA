# SCRA: Symmetric Cross-modal Reasoning Alignment for Unpaired OCT-Fundus Diagnosis

Welcome to the official repository for the paper **Symmetric Cross-modal Reasoning Alignment for Unpaired OCT-Fundus Diagnosis**. This repository provides comprehensive documentation for the ophthalmic imaging datasets utilized in our research and will host the official implementation of the proposed SCRA framework.

## Overview
Unpaired cross-modal learning in medical imaging faces significant challenges due to modality discrepancies and the absence of paired supervision. Our work reformulates this problem as a structured semantic reasoning alignment task. We introduce a unified pipeline that maps heterogeneous fundus and OCT images into a shared concept distribution space, refines semantic representations through graph-based reasoning, and enforces cross-modal consistency via symmetric distribution alignment.

## Dataset Usage and Construction
To evaluate the proposed framework under realistic clinical conditions, we systematically aggregated and processed a large-scale collection of publicly available ophthalmic datasets. The construction pipeline follows these key steps:
- **Data Aggregation:** We collected 67 publicly accessible datasets, comprising 43 color fundus photography collections and 24 optical coherence tomography cohorts.
- **Quality Control and Standardization:** All images underwent rigorous filtering to exclude samples with insufficient resolution, missing annotations, or severe acquisition artifacts. Formats were standardized for uniform processing.
- **Clinical Categorization:** Guided by established diagnostic criteria, the retained samples were consolidated into ten clinically consistent categories. These include normal controls and nine pathological conditions: cataract, diabetic retinopathy, glaucoma, inflammatory or infectious retinal diseases, macular diseases, myopia-related conditions, optic nerve disorders, retinal vascular diseases, and retinopathy of prematurity.
- **Partitioning:** The integrated dataset was divided into training, validation, and test subsets using an 8:1:1 ratio with stratified sampling to preserve category proportions across modalities.

The aggregated dataset contains 346,649 fundus images and 457,812 OCT images, totaling 804,461 samples. Detailed statistics and the complete processing pipeline are available in the `dataset/` directory of this repository. For programmatic access, a machine-readable mapping file (`dataset_mapping.csv`) is also provided.
